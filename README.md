# Язык ассемблера для Учебной Виртуальной Машины (Вариант 13)

Язык ассемблера для УВМ использует YAML-формат для описания программ. Каждая программа представляет собой последовательность команд, каждая из которых состоит из операции (op) и аргумента (arg). Команды выполняются последовательно, регистр-аккумулятор используется как основной рабочий регистр.

__Команда load (загрузка константы)__\
Назначение: Загружает 30-битную константу в регистр-аккумулятор.

Формат: load <константа>

Размер команды: 5 байт

Поля команды:\
Код операции (A): 1 (занимает биты 0-2)\
Константа (B): 30-битное значение (занимает биты 3-32)

__Команда read (чтение из памяти)__\
Назначение: Читает значение из памяти по указанному адресу и помещает его в аккумулятор.

Формат: read <адрес>

Размер команды: 3 байта

Поля команды:\
Код операции (A): 2 (занимает биты 0-2)\
Адрес (B): 20-битный адрес памяти (занимает биты 3-22)


__Команда read (чтение из памяти)__\
Назначение: Читает значение из памяти по указанному адресу и помещает его в аккумулятор.

Формат: read <адрес>

Размер команды: 3 байта

Поля команды:\
Код операции (A): 2 (занимает биты 0-2)\
Адрес (B): 20-битный адрес памяти (занимает биты 3-22)


__Команда write (запись в память)__\
Назначение: Записывает значение из аккумулятора в память по указанному адресу.

Формат: write <адрес>

Размер команды: 3 байта

Поля команды:\
Код операции (A): 7 (занимает биты 0-2)\
Адрес (B): 20-битный адрес памяти (занимает биты 3-22)


__Команда shift_right (побитовый арифметический сдвиг вправо)__\
Назначение: Выполняет побитовый арифметический сдвиг вправо значения в аккумуляторе на количество бит, указанное в памяти по заданному адресу. Результат сохраняется в аккумуляторе.

Формат: shift_right <адрес>

Размер команды: 3 байта

Поля команды:\
Код операции (A): 5 (занимает биты 0-2)\
Адрес (B): 20-битный адрес памяти (занимает биты 3-22)
